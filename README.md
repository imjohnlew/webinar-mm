# AI Agency Business Model Visualizer

A teaching tool for "Day 1" of your AI Agency course. This app helps students visualize and understand the business models of their potential clients (Dental, Lawyer, Coach).

## Features
- **Industry Selection**: Interactive cards to choose a niche.
- **Visual Mapping**: Clear breakdown of Lead -> Funnel -> Offer.
- **Skill Highlighting**: Shows exactly where skills (Video Editing, Automation, Funnel Building) fit into the client's business.
- **Customer Journey**: Visual flow of how a stranger becomes a customer.

## How to Run
1. Navigate to the project directory:
   ```bash
   cd ai-agency-viz
   ```
2. Install dependencies (if not already done):
   ```bash
   npm install
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```
4. Open the link (usually http://localhost:5173).

## Tech Stack
- React
- Vite
- TailwindCSS (Styling)
- Framer Motion (Animations)
- Lucide React (Icons)
